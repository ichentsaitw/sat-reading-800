SECTION 01

為什麼閱讀高分難再突破？

數學腦 vs 閱讀腦——不是你不夠聰明，是用錯了方法

01

用「感覺」選答案

「這個選項聽起來對」——但 SAT 只認文本有寫的，不認邏輯推演。

02

課外知識干擾判斷

讀到熟悉主題（科學、歷史），不自覺用「我知道的」去答題，偏離文章。

03

被「幾乎正確」的選項騙了

出題者特別為 700 分設計了「99% 像正確答案」的陷阱選項。

04

忽略語氣和語境

答案事實正確，但語氣太強或太弱，例如「批評」跟「強烈譴責」差很多。

05

看到熟悉詞就停下來

選項引用文章的詞彙，但套在不同段落、不同語境——直接中招。

⚡

核心心態轉換

不是在找「對的答案」，是在找「無法被文本反駁的答案」。
每次選答案前，要能指出文章裡哪一行話支持你的選擇。
這個思維跟做數學證明題一樣——你需要的是「文本授權」，不是「感覺授權」。

SECTION 02

8 大陷阱題型

認識這 8 種陷阱，就能把「感覺對」變成「確定對」

🎭
TRAP 01
幾乎正確陷阱
Almost Right Trap
用文章出現過的詞彙，但套在錯誤的段落或語境上——製造熟悉感。
⚠ 陷阱長這樣
問題問第 25–32 行的角色動機，陷阱選項卻引用第 18–20 行的「責任感」——詞對，但位置錯。
✅ 破解方法
每次選答案前，必須能指出文章中「具體哪一行」支持你的選擇。找不到那一行 → 不選。
🚫
TRAP 02
極端語言陷阱
Extreme Language Trap
選項包含 always / never / all / must / completely / only 等絕對詞——通常是錯的。
⚠ 陷阱長這樣
作者只是「質疑某些補助計畫」，但陷阱說「應該完全取消，因為永遠導致偏見」。
✅ 破解方法
看到 always / never / must / all → 立刻提高警覺。正確答案通常用 might、could、suggests 等軟化詞。
🔄
TRAP 03
反向陷阱
Opposite Answer Trap
用文章的真實概念，但把主體或方向反轉——快速閱讀最容易中招。
⚠ 陷阱長這樣
文章 A 批評某理論，文章 B 支持它。陷阱說「A 的作者支持 B 的觀點」——直接把立場換了。
✅ 破解方法
讀完每篇文章，立刻用一句話總結立場。比對答案時，逐字確認主體（誰對誰）沒有被對調。
🌐
TRAP 04
超出範圍陷阱
Out of Scope Trap
答案在邏輯上合理，但文章根本沒說這件事——知識越豐富越容易中招。
⚠ 陷阱長這樣
問志願服務的挑戰，陷阱答案說「能強化社區凝聚力」——聽起來對，但文章完全沒提。
✅ 破解方法
黃金原則：如果你無法在文章找到「具體一句話」支持這個選項 → 不選。「我覺得這是對的」不算。
🎭
TRAP 05
語氣不符陷阱
Wrong Tone Trap
答案在事實上準確，但情感強度或語氣比文章原文重太多或太輕。
⚠ 陷阱長這樣
作者「質疑（question）」某理論，陷阱答案說作者「猛烈攻擊（fiercely attacked）」——過度放大。
✅ 破解方法
先判斷作者語氣：正面/負面？強烈/溫和？然後排除語氣不符的選項。學術文章語氣通常偏謹慎。
🎯
TRAP 06
真實但無關陷阱
True But Irrelevant
選項內容在文章中確實存在，但它回答的不是這道題問的問題。
⚠ 陷阱長這樣
問「作者為什麼引用這個實驗」，陷阱答案正確描述實驗結果，但沒說作者引用它的目的。
✅ 破解方法
反覆確認：「這個選項直接回答了題目問的那個問題嗎？」內容正確但答非所問 → 不選。
📖
TRAP 07
最常見定義陷阱
Common Definition Trap
詞彙題選了這個字最常見的意思，而不是在這段文字中的意思。
⚠ 陷阱長這樣
問 "lead" 在段落中的意思，段落在討論鉛筆，正確答案是「石墨」，陷阱答案是「領導/引導」。
✅ 破解方法
詞彙題的正確做法：先遮住選項，用自己的話想一個替換詞，再去找最接近的選項。永遠不要直接看選項。
❌
TRAP 08
忽略否定詞陷阱
Negation Trap
題目有 NOT / EXCEPT / LEAST，學生沒看到，結果選了完全相反的答案。
⚠ 陷阱長這樣
題目問「以下哪個不是作者支持的觀點？」學生選了「作者最強烈支持的那個」——大翻車。
✅ 破解方法
看到 NOT / EXCEPT / LEAST → 立刻用筆圈起來，並在腦中把問法重新翻譯一遍再解題。
SECTION 03

各題型解法

點開每個題型，看專屬解法——每種題都有固定的破關流程

詞彙題
Words in Context ── 語境詞彙題
佔 ~28% ▼

最常見題型，問某個詞在這段文字中最接近什麼意思，或選最合適的詞填空。

  • 遮住選項 — 先不看 A/B/C/D
  • 讀前後文脈 — 不只看含目標詞那句，也看前後句
  • 判斷語氣方向 — 作者這裡想表達正面/負面/中立？強調或弱化？
  • 自己想一個替換詞 — 用你自己的話，想這個位置需要什麼意思
  • 找最接近的選項 — 不問「這個對不對」，而是問「哪個最接近我想的那個詞」

💡 高頻學術詞彙要熟：undermine、advocate、assert、distinguish、nuanced、prevalent、contend、compelling……

證據題
Command of Evidence ── 命令證據題
高頻出現 ▼

通常成對出現：Q1 問一個主張，Q2 問「哪段引文最支持你的 Q1 答案」。

🔁 雙向互鎖法

  • 先回答 Q1，形成答案假設
  • 到 Q2 找能支持你 Q1 答案的引文
  • 如果 Q2 沒有任何引文能支持 → 回 Q1 換選項，重來
  • 找到最直接連結的引文（推論鏈越短越好）

⚠ 常見陷阱：引文來自不相關段落、引文只支持細節但 Q1 問大圖景。

推論題
Inference / Logical Completion ── 推論與邏輯完成題
中頻 ▼

問「根據文章，最合理的推論是什麼」或「空格最符合邏輯的填入是什麼」。

🧊 保守預測法

  • 讀完段落，在空格前停下，問自己：「根據文章邏輯，下一步應該是什麼？」
  • 先形成保守的預測（不推得太遠）
  • 找含 might / could / suggests / indicates 等軟化詞的選項
  • 排除推論過度的選項（文章說「觀察到 X」，不要選「X 必然導致 Y」）
目的題
Text Purpose & Structure ── 文章目的與結構題
中頻 ▼

問「文章的主要目的是什麼」或「第二段對第一段有什麼功能」。

🔠 動詞識別法

  • 讀完整篇，先用一句話說出「作者想說什麼」
  • 注意開頭和結尾——這兩處通常揭示主旨
  • 答案選項通常以動詞開頭：argue、demonstrate、illustrate、challenge、qualify
  • 選最準確描述作者行為的動詞，不選誇大或縮小的
配對段落
Cross-Text Connections ── 雙段落比對題
每模組 2–3 題 ▼

出現兩段不同作者的短文，問「A 會如何回應 B 的主張」或「兩人最可能都同意什麼」。

  • 先看題目：確認問的是「分歧」還是「共識」，帶目標閱讀
  • 讀文章 A，立刻用一句話總結核心立場
  • 讀文章 B，立刻用一句話總結核心立場
  • 分歧題 → 找「A 的主張直接衝突 B 說的某個說法」
    共識題 → 找「兩人都能接受的最保守陳述」

⚠ 最常見陷阱：把兩位作者的立場對調，或用極端語言「完全反對」「絕對支持」。

SECTION 04

800 分技術

這些不是讀書技巧，是解題協議——每次都要執行，不能憑感覺

🔮
預測答案法
Predict Before Looking
讀完段落後，先遮住選項，用自己的話說出答案應該是什麼。帶著預測再去比較選項，不會被陷阱帶走。
🗑️
反向淘汰法
Elimination First
思維轉換：不是「哪個是對的」，而是「我能把哪三個消除掉」。對每個錯誤選項，必須說出被消除的具體理由。
📍
文本驗證協議
Text Verification
每次選定答案前問自己：「我能指出文章哪一行證明這個答案嗎？」找不到那一行 → 不選。
🔬
限定詞精確度
Qualifier Precision
卡在兩個選項時，看微小限定詞：some vs most vs all、suggests vs proves vs claims、may vs will vs must。這些小詞決定一切。

⏱ 時間管理——每題平均 71 秒（32 分鐘 ÷ 27 題）

詞彙題（短文）
45–60秒
推論題（較長段落）
75–90秒
Cross-Text（兩篇文章）
90–120秒

💡 卡住就跳過原則：一題讀了兩遍還不確定 → 用 Bluebook ↗ 的 flag 標記，先猜一個，繼續往後，最後剩時間再回來。不確定的決定不要花超過 40 秒。

SECTION 05

錯題本怎麼建

研究顯示，有系統分析錯誤的學生，進步幅度比單純重刷高出 13%

+13%
有錯題本 vs 只刷題
分數提升差距
3次
同一類型出現 3 次
→ 下週集中練習這個弱點

錯誤分類系統——每次記錄必須精確分類，不能只寫「不懂」

A 類：技術錯誤

操作層面的失誤

  • 沒回原文，憑記憶回答
  • 沒讀完選項的最後一個字
  • 沒注意 NOT/EXCEPT
  • 詞彙語境 vs 日常意義搞混
B 類：陷阱中招

被設計好的陷阱騙了

  • 被極端語言蒙騙
  • 被段落迴聲（原文照抄）吸引
  • 被個人知識帶走
  • 因果/主客方向反了
C 類：理解問題

閱讀理解層面的問題

  • 沒理解作者語氣
  • 段落功能沒理解
  • 代詞指代搞混
  • 複雜句型邏輯關係誤解
D 類：時間問題

考試節奏出問題

  • 時間不夠，沒認真讀就猜
  • 題目問的和以為的不一樣
  • 在難題卡太久，後面來不及

SECTION 07

備考四階段

不管你什麼時候開始，這四個階段都要走一遍

階段一

🔍 診斷——先弄清楚自己錯在哪

  • 在 Bluebook ↗ 完成 1 套完整練習測驗（有計時）
  • 詳細分析所有錯題，建立錯題本
  • 分類所有錯誤，找出主要弱點題型（通常 2–3 種）
  • 沒做過診斷就開始刷題，等於瞎打靶
階段二

🎯 集中——針對弱點題型專攻

  • 針對弱點題型計時練習，不要混著全部一起練
  • 每次練完：複習錯題本，填寫「下次要怎麼做」
  • 穿插無計時深度複習，理解每個錯誤的根本原因
  • 同一類型犯錯超過 3 次，就要專門研究那個陷阱型
階段三

🔄 整合——模擬真實考試節奏

  • 混合練習，不再只練特定題型
  • 定期完成一套完整測驗（兩個模組，計時）
  • 持續更新錯題本，確認弱點是否縮小
階段四

🚀 衝刺——考前保持狀態

  • 多做完整模擬測驗（計時、正式環境）
  • 重點複習錯題本中重複出現的問題
  • 確認每種陷阱型都能識別並對抗
  • 考前最後幾天：輕量複習，不要刷新題，保持狀態
